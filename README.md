## About The Project

K4-DropHostages is a Counter-Strike 2 plugin that removes the restriction on where players can drop hostages. By default, CS2 only allows dropping hostages in specific rescue zones. This plugin enables players to drop hostages anywhere on the map, providing more tactical flexibility in hostage rescue scenarios.

> **✨ When used with hooks, the feature feels like a natural part of the game!** The seamless integration makes dropping hostages anywhere feel like it was always meant to be there.

### Dependencies

To use this server addon, you'll need the following dependencies installed:

- [**CounterStrikeSharp**](https://github.com/roflmuffin/CounterStrikeSharp/releases): CounterStrikeSharp allows you to write server plugins in C# for Counter-Strike 2/Source2/CS2.
